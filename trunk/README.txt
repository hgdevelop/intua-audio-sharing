INSTRUCTIONS
============

1. Open the project with XCode, you need at least iPhone SDK 3.0
2. Open the UIPasteboardAudio-Info.plist file located under "Resources" group
3. Edit the "Bundle Identifier" field to match your company name / code signing identity.
4. Close the plist editor and save the file
5. -OPTIONAL- Under menu Project > Edit Project Settings, adapt the "Code Signing Identity"
for "Any iPhone OS Device".
